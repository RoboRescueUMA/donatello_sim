Primera isntalación hacer también:
$ sudo apt-get install liburdfdom-tools

    Para editar donatello.urdf :
        $ code ~/catkin_ws/src/donatello_sim/urdf/donatello.urdf

    Para subir la configuración de donatello.urdf:

        (Proyecto anterior)
            EJECUTAR ESTA LINEA ANTES DE HACER catkin_make
            $ rosrun xacro xacro ~/catkin_ws/src/donatello_sim/urdf/donatello.urdf.xacro > ~/catkin_ws/src/donatello_sim/urdf/donatello.urdf

        $ rosparam set /robot_description -t ~/catkin_ws/src/donatello_sim/urdf/donatello.urdf
        
        Esto ajusta los parámetros de donatello.urdf a RobotModel dentro de RViz, por lo que:
        $ roscore
        $ rviz
        Dentro de rviz:
            Pulsar "Add" en el menú de la izquierda.
            Seleccionar "RobotModel".
